Redis

Redis — нереляционное in-memory хранилище данных, работающее по принципу «ключ‑значение». Часто используется как кэш, брокер сообщений и для хранения сессий.

Возможности

  • Поддержка множества структур данных: строки, списки, множества, хэши, потоки
  • Встроенная репликация и кластеризация
  • Pub/Sub для обмена сообщениями
  • Персистентность на диск (RDB, AOF)

Redis обеспечивает задержки менее миллисекунды и незаменим в высоконагруженных веб-приложениях для кэширования результатов запросов.
