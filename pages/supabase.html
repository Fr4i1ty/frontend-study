Supabase

Supabase — это открытая облачная платформа, которая предоставляет базу данных PostgreSQL, аутентификацию, файловое хранилище и real‑time API из коробки.

Что предлагает Supabase

  • Полноценный PostgreSQL с веб‑консолью
  • Встроенная аутентификация (email, соцсети)
  • Realtime подписки через WebSockets
  • Автоматически генерируемое REST API

Supabase ускоряет разработку современных веб‑приложений, заменяя связку Firebase + база данных на открытом стеке.
