О проекте

Учебный проект в рамках курса "Технологии разработки интернет-приложений". Цель — закрепить навыки создания семантической, адаптивной и доступной вёрстки с применением современного CSS‑фреймворка и веб‑компонентов.

Тематика

Обзор и сравнение популярных баз данных, используемых в веб‑разработке: PostgreSQL, MySQL, MongoDB, Redis и Supabase.

Использованные технологии

  • HTML5, CSS3
  • CSS‑фреймворк: Chota
  • JavaScript (ES6+) – веб‑компоненты
  • Git и GitHub Pages

Контент страниц основан на открытых источниках и документации соответствующих продуктов.
